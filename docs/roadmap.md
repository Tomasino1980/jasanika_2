# Roadmap projektu

## M0 - Initialize

Cíl:

* vytvoření repozitáře
* vytvoření dokumentace
* definice pravidel projektu
* inicializace Git workflow

Výstup:

* připravený základ projektu

---

## M1 - Core Foundation

Cíl:

* bootstrap systému
* constants
* autoloading
* základní inicializace frameworku

Výstup:

* funkční Core vrstva

---

## M2 - Service Container

Cíl:

* dependency injection container
* registrace služeb
* resolving služeb

Výstup:

* funkční DI Container

---

## M3 - Module System

Cíl:

* registrace modulů
* lifecycle modulů
* aktivace/deaktivace modulů

Výstup:

* modulární architektura

---

## M4 - Configuration System

Cíl:

* konfigurace frameworku
* načítání nastavení
* validace konfigurace

Výstup:

* centralizovaný config systém

---

## M5 - Asset Manager

Cíl:

* CSS registry
* JS registry
* enqueue systém
* verzování assetů

Výstup:

* správa frontend assetů

---

## M6 - Hook Manager

Cíl:

* wrapper pro WordPress hooky
* registrace akcí
* registrace filtrů

Výstup:

* jednotná práce s WP hooky

---

## M7 - Admin Foundation

Cíl:

* základ administrace
* menu systém
* stránková architektura

Výstup:

* admin framework

---

## M8 - WooCommerce Foundation

Cíl:

* WooCommerce integrace
* wrapper služby
* kompatibilita

Výstup:

* základ e-shop vrstvy

---

## M9+

Další moduly budou definovány samostatně podle potřeb projektu.
