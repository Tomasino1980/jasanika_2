- Dynamic theme settings CSS injection via ThemeSettingsCompiler
- Frontend rendering via ThemeRenderer with layout templates
- Archive grid, search results, 404 page, single post/page templates
- Responsive container with dynamic container width
- Navigation menus with dropdown support
- Responsive logo V3 (desktop, mobile, retina) with size/position controls
- Sticky header with scroll detection
- Search toggle (desktop inline, mobile fullscreen)
- CTA Button with component framework integration

- Footer with widget areas, navigation, copyright
- Hero section (static and slider modes)
