# Typography System

## Důležité pravidlo

Používej pouze definované fonty.

Nepřidávej další fonty bez schválení.

---

# Heading Font

Font Family:

Playfair Display

Použití:

* H1
* H2
* H3
* Hero sekce
* Bannery

Váhy:

* 600
* 700

# Body Font

Inter

Použití:

* text

Váhy:

* 400
* 500
* 600
